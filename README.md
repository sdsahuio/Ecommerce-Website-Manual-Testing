# E-commerce Website Manual Testing Project



## Project Overview

This repository contains a complete manual testing project for an e-commerce website.



The project includes:

- Test Plan

- Test Cases

- Bug Reports

- Requirement Traceability Matrix (RTM)

- Screenshots



## Modules Tested

- Login

- Registration

- Search

- Add to Cart

- Checkout

- Payment



## Testing Types Performed

- Functional Testing

- Regression Testing

- Smoke Testing

- UI Testing

- Usability Testing



## Tools Used

- Microsoft Excel

- Microsoft Word

- Jira (Demo)

- Google Chrome
