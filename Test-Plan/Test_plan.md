# TEST PLAN

## Project Name
E-commerce Website Testing

## Objective
To verify that all functionalities of the e-commerce application work correctly.

## Scope
Testing includes:
- Login
- Registration
- Product Search
- Add to Cart
- Checkout
- Payment

## Testing Types
- Functional Testing
- Smoke Testing
- Regression Testing
- UI Testing

## Entry Criteria
- Application URL available
- Test environment ready

## Exit Criteria
- All critical defects fixed
- Test cases executed successfully

## Risks
- Server downtime
- Payment gateway unavailable

## Deliverables
- Test Cases
- Bug Reports
- RTM
- Test Summary
